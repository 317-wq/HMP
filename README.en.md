# 高并发内存池

#### Description
HMP（High-concurrency Memory Pool）是 一个面向高并发场景的 C++ 内存池实现。
采用线程本地缓存 + 细粒度锁策略，显著降低多线程内存分配竞争。
支持高效的对象分配/释放、内存对齐、零碎片化设计，适用于游戏服务器、网络服务、高并发中间件等场景。
本项目为学习与实践性质，代码结构清晰，注释完整，便于二次开发与性能调优。
